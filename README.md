# Tewodros Endalamaw — Portfolio

Static personal website (GitHub Pages friendly) built with HTML, CSS, and JavaScript.

## Edit content
- Main page: `index.html`
- Styles: `styles/styles.css`
- Scripts (theme, language, modal, filters): `scripts/main.js`

## Run locally
Any static server works. Example (PowerShell):
```powershell
python -m http.server 8000
```
Then open `http://localhost:8000`.
